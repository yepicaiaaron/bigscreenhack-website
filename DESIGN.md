# Big Screen Hack - Cannes Edition
## Design Document

### Project Overview

**Website**: www.bigscreenhack.com  
**Event**: Big Screen Hack - Cannes Film Festival Edition  
**Tagline**: "Hack Your Way to the Big Screen in 24 Hours"  
**Tool**: VideoStack - AI-powered storytelling without the prompting pain

### Core Concept: "Digital Rebellion: The Cannes Override"

The website is an interactive, scroll-based "power lapse" experience that takes users on a journey from digital chaos to cinematic triumph. It showcases:
- The last Big Screen Hack event (BFI Premiere winners)
- The upcoming Cannes Film Festival events
- VideoStack's creative toolkit
- The 24-hour hack challenge

### Design Principles

1. **Digital Rebellion Aesthetic**
   - Raw, glitch-inspired visuals
   - Neon accents on black backgrounds
   - Intentional digital imperfections
   - Cyber-brutalist elements

2. **Scroll-Based Power Lapse**
   - Each scroll triggers dramatic animations
   - Sections transition with glitch effects
   - Parallax layers create depth
   - Interactive elements respond to scroll position

3. **Extreme Art Styles**
   - Avoid cliché AI-generated looks
   - Bold, unconventional color palettes
   - Mixed media and digital collage
   - Distressed textures and grunge elements

4. **Immersive Storytelling**
   - The website itself is a narrative experience
   - From "System Failure" to "Cannes Override"
   - VideoStack as the hero tool
   - User as the protagonist

### Scroll Animations (Power Lapse)

**Global Scroll Effects:**
- Progress bar with glitch aesthetic
- Section indicators as circuit nodes
- Background noise intensity tied to scroll speed
- Parallax depth on all layers

**Section Transitions:**
1. **Glitch Wipe**: Digital noise transition
2. **Pixel Sort**: Content dissolves into pixels
3. **Scan Line Reveal**: CRT-style section reveal
4. **Data Moshing**: Chaotic but controlled transition
5. **System Override**: Clean wipe with digital artifacts

**Micro-Interactions:**
- Hover: Glitch intensifies
- Click: Digital "thump" effect
- Scroll: Parallax layers shift
- Load: Progressive reveal with noise

### Technical Implementation

**Libraries:**
- GSAP (GreenSock) for animations
- ScrollTrigger for scroll-based effects
- Three.js (optional) for 3D elements
- Custom shaders for glitch effects

**Performance:**
- Lazy loading for images
- Optimized animations (GPU-accelerated)
- Reduced motion support
- Mobile-responsive fallbacks

**Accessibility:**
- Keyboard navigation
- Screen reader support
- High contrast mode
- Pause animations option
