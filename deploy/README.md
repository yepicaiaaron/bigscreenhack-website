# Big Screen Hack - Cannes Edition

**Live Site:** https://bigscreenhack.netlify.app

An interactive, scroll-based "power lapse" website showcasing the Big Screen Hack event and upcoming Cannes Film Festival events.

## About

The Big Screen Hack is a 24-hour creative marathon where filmmakers, developers, and storytellers "hack their way to the big screen" using VideoStack—a revolutionary tool that removes the pain of prompting and unleashes creativity.

## Features

- ⚡ Scroll-based power lapse navigation
- 🎨 Digital Rebellion aesthetic with glitch effects
- 🎬 7 interactive sections
- ⏱️ Live countdown to Cannes
- 📱 Fully responsive
- 🚀 Deployed on Netlify

## Sections

1. **System Failure Boot** - Glitchy entry point
2. **The Big Screen Hack** - Bold neon title
3. **Last Event** - BFI Premiere showcase
4. **VideoStack Tools** - Creative toolkit
5. **24-Hour Hack** - Timeline & countdown
6. **Cannes Film Festival** - Upcoming events
7. **Enter the Hack** - Registration

## Tech Stack

- HTML5
- CSS3 (no frameworks)
- Vanilla JavaScript
- Netlify Hosting

## Local Development

Simply open `index.html` in any browser. No build process needed!

## Deployment

This site is configured for instant deployment to Netlify:
- Drag & drop the `deploy/` folder to https://app.netlify.com/drop
- Or connect to GitHub for continuous deployment
