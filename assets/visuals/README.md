### Visual Style Guide

All visuals follow the "Digital Rebellion" aesthetic:

- **Background:** Pure black (#000000)
- **Accents:** Neon yellow, hot pink, cyan, orange, green
- **Effects:** Glitch, digital noise, scan lines, pixel sorting
- **Typography:** Bold, blocky, distressed
- **Style:** Raw, gritty, cyber-brutalist

### Usage

Reference visuals in HTML/CSS using relative paths:

```html
<img src="assets/visuals/bigscreenhack_dr_visual_1_2026-03-10-01-01-00.png" alt="System Failure">
```

```css
background-image: url('assets/visuals/bigscreenhack_dr_visual_1_2026-03-10-01-01-00.png');
```
