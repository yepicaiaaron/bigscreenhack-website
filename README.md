# Big Screen Hack - Cannes Edition

## Digital Rebellion: The Cannes Override

An interactive, scroll-based "power lapse" website showcasing the Big Screen Hack event and upcoming Cannes Film Festival events.

### Concept

The Big Screen Hack is a 24-hour creative marathon where filmmakers, developers, and storytellers "hack their way to the big screen" using VideoStack—a revolutionary tool that removes the pain of prompting and unleashes creativity.

### Design Philosophy

- **Digital Rebellion Aesthetic**: Raw, glitch-inspired visuals with neon accents
- **Scroll-Based Power Lapse**: Dynamic animations triggered by user scrolling
- **Extreme Art Styles**: Avoiding cliché AI visuals with unique, bold designs
- **Immersive Experience**: Creating a visceral journey from system failure to Cannes triumph

### Tech Stack

- HTML5
- CSS3 (with custom animations)
- JavaScript (GSAP for scroll animations)
- Responsive design

### Sections

1. **System Failure Boot** - The glitchy entry point
2. **The Big Screen Hack Title** - Bold, neon-outlined typography
3. **Last Event Showcase** - BFI Premiere winners and highlights
4. **VideoStack Tools** - The creative toolkit
5. **24-Hour Hack Concept** - The challenge explained
6. **Cannes Film Festival Preview** - Upcoming events
7. **Enter the Hack** - Call to action

### Visual Assets

All visuals follow the "Digital Rebellion" aesthetic with:
- Black backgrounds
- Neon accents (electric yellow, hot pink, cyan, orange)
- Glitch effects and digital noise
- Raw, textured elements
